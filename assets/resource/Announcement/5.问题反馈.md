## 🐞 提交Bug标准流程

1. **必含材料**：  
   - 问题现象 + 复现步骤  
   - `debug/maa.log`（按[常见问题](04_常见问题.md)指南导出）  
   - 问题界面截图  
   - 环境信息（系统/游戏/软件版本）  
2. **提交方式**：  
   - GitHub Issue 标题加 `[BUG]` 前缀  
   - 或群内@管理员or群主（附上述材料）
