# 🚀快速开始

## 三步极速启动

1. **下载与更新**：  
   - [GitHub Releases](https://github.com/miaojiuqing/MAA_bbb/releases)  
   - [mirror酱](https://mirrorchyan.com/zh/projects?source=maafw-badge&rid=Maa_bbb)  
   - [进入Q群下载](https://qm.qq.com/q/XrFQKgGvaI)  

2. **设置**：游戏分辨率设为 **16:9** → 关闭「设置→辅助→显示菜单提示」  
3. **运行**：右键以管理员身份运行 → 选择对应资源（PC选"桌面端"）  

## 重要通知
