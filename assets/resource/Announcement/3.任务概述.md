# 🧩 任务功能概览

> 💡 **使用前必读**：  
>
> - PC端需 **管理员运行** + **16:9分辨率** + 关闭「设置 → 辅助 → 显示菜单提示」  
> - 模拟器仅支持MuMu/雷电（部分功能未适配，详见各任务说明）  
> - 自动战斗运行时 **切勿按Win/Ctrl/Alt等特殊键**，避免系统弹窗干扰  

## 🌅 日常任务（每日自动完成）

| 任务类别       | 核心功能                                                                 | 注意事项                                  |
|:--------------:|:------------------------------------------------------------------------:|:-----------------------------------------:|
| **登录启动**   | 自动登录、月卡领取、防误触、维护检查                                     | 支持二维码登录；PC端需提前关闭菜单提示    |
| **资源领取**   | 日常/凭证任务奖励、邮件领取、家园金币/体力                               | 凭证奖励需提前确认是否氪通行证            |
| **远征派遣**   | 材料远征、家园远征、家园打工                                             | 支持“体力不足吃药”；打工遇S+可继续派遣    |
| **舰团事务**   | 每日委托（自动补货）、保税仓库申请（自定义物品）                         | 委托材料需提前备好                        |
| **其他日常**   | 模拟作战室/拓境战幕减负、冒险委托（自定义地图/报酬优先级）、每日签到     | 冒险委托PC端默认后崩2地图，需手动切地图   |

## 📅 周常任务（每周自动完成）

| 任务           | 核心功能                                                                 | 注意事项                                  |
|:--------------:|:------------------------------------------------------------------------:|:-----------------------------------------:|
| **万象虚境**   | 自定义层数/副本/通行证/次数；6-7层支持减负                               | 通行证消耗建议提前设置；推荐识之律者女武神|
| **记忆战场**   | 减负模式 + 尝试打最后一层                                                | 减负后建议手动打一把保下周资格            |
| **超弦空间**   | 预设编队、战斗逻辑（鏖灭/轮椅配队）                                      | ⚠️ **PC端专属**；非老登慎用；模拟器未适配 |
| **每周活动**   | 答题减负、分享、吼姆秘宝、90张虚境挑战卷购买                             | 分享任务每次运行均会点击                  |
| **乐土商店**   | 智能购买休假卷/镜面/时序通行证                                           | 建议打完乐土再用；不稳定时请关闭          |

- 📦 **乐土商店**：每周智能筛选（详细见 [往世乐土](07_往世乐土.md)）

## ⚡ 其他功能

| 功能             | 说明                                                                     | 限制/提示                                |
|:----------------:|:------------------------------------------------------------------------:|:----------------------------------------:|
| **自动战斗**     | 通用逻辑 / 薇塔流 / 爱愿妖精流 / 崩落符普攻流                            | ⚠️ 模拟器延迟高慎用；PC端推荐            |
| **剧情处理**     | 自动跳过 / 手动选选项 / 自动选首项 / 古早剧情支持                        | 可设置“收藏CG时忽略跳过”                 |
| **圣痕洗词缀**   | OCR识别 + 正则匹配（示例见下方说明）                                     | ⚠️ 识别可能出错；需在指定界面启动        |
| **退出游戏**     | 任务结束后自动关闭游戏                                                   | 软件设置中可配置“运行结束关闭软件”       |

## ⚠️ 全局重要提示（必读！）

| 风险点               | 正确操作                                                                 |
|:--------------------:|:------------------------------------------------------------------------:|
| **战斗中强制停止**   | 克莱茵打灰等任务战斗途中停止会导致游戏死机（属正常现象），请等待战斗结束 |
| **无限刷取设置**     | 重复次数填 `-1` 或超大数字（如 `9999`）                                  |
| **减负模式刻印**     | 克莱茵打灰减负模式需**提前手动选好刻印**，否则系统随机选择               |
| **女武神预设**       | 所有需指定女武神的任务（浅层序列等），请在游戏内提前设置好               |
| **截图参数**         | PC端建议优先选择 `PrintWindow`（设置 → 连接设置）                        |

> 🌟 **提示**：任务默认开启状态已优化，新手建议保留默认配置。高级参数请在软件内查看选项说明，或加入 [QQ交流群](https://qm.qq.com/q/XrFQKgGvaI) 咨询！
