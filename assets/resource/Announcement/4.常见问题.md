# ❓ 常见问题

## 🔑 通用问题

### Q：启动提示"资源加载失败"？

✅ 解决方案：  

1. 删除安装目录下除 `config` 外所有文件夹  
2. 重新解压最新版到原目录  
3. 重启软件  
💡 持续报错请等待30分钟内修复更新  

### Q：渠道服资源缺失？

✅ 提供包名：  

- 手机：设置 → 应用 → 崩坏3 → 查看包名  
- 模拟器：QQ群内发送APK给管理员  

## 💻 PC端问题

### Q：无法识别游戏窗口？

✅ 四步排查：  

1. 是否管理员运行？  
2. 游戏是否16:9分辨率？  
3. 是否关闭"显示菜单提示"？（见配置指南图示）  
4. 软件内"截图测试"是否正常？  

## 📱 模拟器问题

### Q：连接后无响应？

✅ 检查清单：  

- 模拟器分辨率设为16:9  
- 重启模拟器 + 软件  
- 软件内执行"连接测试"  
- 优先使用MuMu/雷电模拟器  

## 🛠️ 技术支持

### Q：如何导出有效日志？

✅ 黄金流程：  

1. 删除 `debug` 文件夹  
2. 复现问题  
3. 点击软件内 **|→** 按钮导出日志  
4. 连同截图提交至[问题反馈](05_问题反馈.md)  

### Q：双击无法启动，命令行可运行？

✅ Windows安全策略解除：  
右键 `MFAAvalonia.exe` → 属性 → 勾选"解除锁定" → 应用  

## ⚠️ 安全须知

### Q：会被封号吗？

✅ 说明：  

- 基于图像识别，**不修改游戏文件/内存**  
- 同类开源项目**无封号案例**  
- 但无法100%保证，竞技类玩法（记忆战场等）请勿使用  
- 本工具仅限单机内容辅助
