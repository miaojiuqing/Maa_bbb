## 🌱 克莱茵打灰（PC端专属）

**功能定位**：自动化刷取往世乐土克莱茵关卡，高效提升装甲等级、获取乐土币等资源。

### 三种模式详解

| 模式         | 核心逻辑                     | 适用场景               | 关键配置项（软件内设置）                     |
|:------------:|:----------------------------:|:----------------------:|:--------------------------------------------:|
|**减负模式刷**| 自动完成减负流程（刻印需手动选）| 日常稳定刷取           | 走路参数 / 战斗缓冲 / 乐土币满停 / 升满级停 |
| **首层复刷** | 无限循环刷取第一层            | 快速获取基础资源       | 走路参数 / 战斗缓冲 / 乐土币满停 / 升满级停 |
| **浅层序列** | 按序列刷取浅层（支持女武神选择） | 有目标刷取特定资源     | 女武神选择 / 试用角色 / 难度 / 升级策略     |

### ⚠️ 重要须知（必读！）

| 项目         | 说明                                                                 |
|:------------:|:--------------------------------------------------------------------:|
| **平台限制** | ✅ 目前仅支持 PC 客户端（模拟器暂未适配，会尽快适配，欢迎催更！）    |
| **战斗中止** | ⚠️ 战斗途中强制停止会导致游戏死机（属正常现象），请等待战斗结束后操作|
| **无限刷取** | 🔁 重复次数填 `-1` 或超大数字（如 `9999`）即可无限循环               |
| **减负模式** | 🎯 刻印需**提前手动选好**（未选则系统随机）；女武神需**游戏内预设**  |
| **智能停止** | 🛑 支持“乐土币刷满后停止”、“升满级后停止”（需在选项中开启）          |

### 💡 使用指南

1. **基础操作**  
   - 任务入口：选择「克莱茵乐土打灰」  
   - 模式选择：在“要用什么刷等级”中三选一  
   - 重复次数：日常填 `1-3`；无限刷填 `-1`  

2. **减负模式 / 首层复刷专属**  
   - 走路参数：调整“每次前进走多久”、“间隔”、“尝试次数”（默认已优化）  
   - 战斗缓冲：避免因帧率波动导致识别失败  

3. **浅层序列专属**  
   - 女武神：指定角色或启用“试用女武神”  
   - 难度：自动匹配当前可挑战最高难度  
   - 升级：开启“可升级时主动升级”提升效率
